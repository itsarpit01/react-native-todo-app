## 📲 BUILDING APK

### Method 1: EAS Build (Recommended)
```bash
npm install -g eas-cli
eas login
eas build --platform android --build-type apk
```
Time: 5-10 minutes | Effort: Easy | Cost: Free tier available

### Method 2: Local Build
```bash
npx expo prebuild --clean
npm run android
```
Time: 10-20 minutes | Effort: Medium | Cost: Free

### Method 3: Android Studio
```bash
npx expo prebuild --clean
open -a "Android Studio" android/
# Build → Build APK(s)
```
Time: 15-25 minutes | Effort: Medium | Cost: Free

See **APK_BUILD_GUIDE.md** for detailed instructions.
